01.数组中重复的数字

在一个长度为 n 的数组 nums 里的所有数字都在 0～n-1 的范围内。数组中某些数字是重复的，但不知道有几个数字重复了，也不知道每个数字重复了几次。请找出数组中任意一个重复的数字。

利用count数组存0~n-1出现的次数, 当其>1时说明是重复元素

public int findRepeatNumber(int[] nums){
    int[] count = new int[nums.length];
    for(int n:nums){
        count[n]++;
        if(count[n]>1) return n;
    }

    return -1;

}

public boolean findNumberIn2DArray(int[][] matrix, int target) {
    // 判空
    if(matrix==null||matrix.length==0||matrix[0].length==0) return false;

    int m = 0;
    int n = matrix[0].length-1;
    while(m<matrix.length&&n>=0){
        // 如果==, 返回
        if(matrix[m][n]==target){
            return true;
        }
        // 如果<target, 说明第m行都比target小
        else if(matrix[m][n]<target){
            m++;
        }
        // 如果>target, 说明第n列都比target大
        else{
            n--;
        }
    }
    return false;
}

public String replaceSpace(String s) {

    if(s==null||s.length()==0) return s;
    // 转成char数组
    char[] ch = s.toCharArray();
    char[] res = new char[3*ch.length];
    int index=0;
    for(int i=0;i<ch.length;i++){
        if(ch[i]==' ') {
            res[index++]='%';
            res[index++]='2';
            res[index++]='0';
        }
        else{

        }
    }
    return new String(res,0,index);
}

public int[] reversePrint(ListNode head) {
    int len = 0;
    ListNode cur = head;
    // 先求出链表长度
    while(cur!=null){
        len++;
        cur = cur.next;
    }
    // 从后往前装入数组
    int[] res = new int[len];
    cur = head;
    while(cur!=null){
        res[--len] = cur.val;
        cur = cur.next;
    }
    return res;
}

05.重建二叉树

输入某二叉树的前序遍历和中序遍历的结果，请构建该二叉树并返回其根节点。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。

给出前序,中序, 重建树
递归, 按照前序顺序重建节点 dfs(pre,in,finish)
递归终止条件: 按前序已建立好所有节点,或者到了中序的截断点

// 前序
int preindex;
// 中序
int inindex;
public TreeNode buildTree(int[] preorder, int[] inorder) {
    preindex = 0;
    inindex = 0;
    return dfs(preorder,inorder,null);
}
// pre:前序 in:中序 finish:分段截断点
// 功能：重建finish节点前的所有节点
TreeNode dfs(int[] pre,int[] in,TreeNode finish){
    // 递归终止条件
    // 如果按照前序遍历已经重建好了所有节点，或者到了中序里的截断点
    if(preindex==pre.length||(finish!=null&&finish.val==in[inindex])) return null;

    // 重建根节点: 前序遍历
    TreeNode r = new TreeNode(pre[preindex]);
    preindex++;

    // 左 递归 中序遍历r之前的节点
    r.left = dfs(pre,in,r);
    inindex++;

    // 右
    r.right = dfs(pre,in,finish);

    return r;
}

06.用两个栈实现队列

用两个栈实现一个队列。队列的声明如下，请实现它的两个函数 appendTail 和 deleteHead ，分别完成在队列尾部插入整数和在队列头部删除整数的功能。(若队列中没有元素，deleteHead 操作返回 -1 )

添加时A直接存入, 删除时B中有元素则从B中弹出, 没有则将A中的全部元素弹入B中, 再弹出B顶

class CQueue {
    Stack<Integer> A;
    Stack<Integer> B;

    public CQueue() {
        A = new Stack<>();
        B = new Stack<>();
    }

    public void appendTail(int value) {
        // 添加时直接添加
        A.push(value);
    }

    public int deleteHead() {
        // 删除时倒到B中删除，B作为一个缓冲区使用
        // 如果B中此时有值，直接从B中返回
        if(!B.isEmpty()) return B.pop();
        else{
            while(!A.isEmpty()){
                B.push(A.pop());
            }
            if(!B.isEmpty()) return B.pop();
        }

        return -1;
    }
}

找临界点: 二分

class Solution {
    public int minArray(int[] numbers) {
        //找临界点：二分
        //1. 没有翻转的情况
        int l=0,r=numbers.length-1;
        if(numbers[0]<numbers[r]) return numbers[0];
        //2. 3451222的情况
        while(r>0&&numbers[r]==numbers[r-1]) r--;

        //二分
        int p = numbers[0];
        int mid = 0;
        while(l<r){
            mid = l+(r-l)/2;
            if(numbers[mid]>=p) l=mid+1;
            if(numbers[mid]<p) r =mid;
        }
        return numbers[r];
    }
}

09.矩阵中的路径

给定一个 m x n 二维字符网格 board 和一个字符串单词 word 。如果 word 存在于网格中，返回 true ；否则，返回 false 。单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中“相邻”单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母不允许被重复使用。

深度优先二维搜索 dfs(board,word,i,j,s)

class Solution {
    int m;
    int n;
    public boolean exist(char[][] board, String word) {
        //深度优先
        if(board==null||board.length==0||board[0].length==0) return false;
        m=board.length;
        n=board[0].length;
        for(int i=0;i<m;i++){
            for(int j=0;j<n;j++){
                if(dfs(board,word,i,j,0)) return true;
            }
        }
        return false;
    }
    //b:字母矩阵 word:单词 i,j 起点 s:单词检索索引
    boolean dfs(char[][] b,String word,int i,int j,int s){
        //递归终止条件

        //越界或者不等 返回false
        if(i<0||i>=m||j<0||j>=n||b[i][j]!=word.charAt(s)) return false;
        //如果上面没有返回false且word全走完了 返回true
        if(s==word.length()-1) return true;
        char temp = b[i][j];
        //利用'*'代替visited标记
        b[i][j] = '*';
        //搜索上下左右
        boolean res = dfs(b,word,i+1,j,s+1)
            ||dfs(b,word,i-1,j,s+1)
            ||dfs(b,word,i,j+1,s+1)
            ||dfs(b,word,i,j-1,s+1);

        //回溯
        b[i][j] = temp;

        return res;
    }
}

10.机器人运动范围

地上有一个m行n列的方格，从坐标 [0,0] 到坐标 [m-1,n-1] 。一个机器人从坐标 [0, 0] 的格子开始移动，它每次可以向左、右、上、下移动一格（不能移动到方格外），也不能进入行坐标和列坐标的数位之和大于k的格子。例如，当k为18时，机器人能够进入方格 [35, 37] ，因为3+5+3+7=18。但它不能进入方格 [35, 38]，因为3+5+3+8=19。请问该机器人能够到达多少个格子？

dfs(x,y,xh,yh)

class Solution {
    //visited记录已经访问的位置
    boolean[][] visited;
    int target;
    int M;
    int N;
    public int movingCount(int m, int n, int k) {
        //深度优先
        M=m;
        N=n;
        target = k;
        visited = new boolean[M][N];
        //传入起点, x的位和xh,y的位和yh
        return dfs(0,0,0,0);
    }
    //x,y 起点 xh:x位和 yh:y位和
    int dfs(int x,int y,int xh,int yh){
        //终止条件:越界||已访问||超过规定k
        if(x<0||x>=M||y<0||y>=N||visited[x][y]||(xh+yh)>target) return 0;
        visited[x][y] = true;
        //19->20 和小了8
        //18->19 和大了1
        return 1+ dfs(x+1,y,(x+1)%10==0?xh-8:xh+1,yh)
            +dfs(x,y+1,xh,(y+1)%10==0?yh-8:yh+1);
    }
}

11.剪绳子

给你一根长度为 n 的绳子，请把绳子剪成整数长度的 m 段（m、n都是整数，n>1并且m>1），每段绳子的长度记为 k[0],k[1]…k[m-1] 。请问 k[0]k[1]…*k[m-1] 可能的最大乘积是多少？例如，当绳子的长度是8时，我们把它剪成长度分别为2、3、3的三段，此时得到的最大乘积是18。

动态规划

class Solution {
    public int cuttingRope(int n) {
        //动态规划
        //dp[i]=max(dp[i],dp[i-j]*j,(i-j)*j)
        if(n<1) return 0;
        //1. 初始化dp
        int[] dp = new int[n+1];
        dp[1] = 1;
        //2. 遍历更新
        for(int i=2;i<=n;i++){
            for(int j=1;j<i;j++){
                dp[i] = Math.max(Math.max(dp[i-j]*j,(i-j)*j),dp[i]);
            }
        }
        return dp[n];
    }
}

12.剪绳子(取模)

给你一根长度为 n 的绳子，请把绳子剪成整数长度的 m 段（m、n都是整数，n>1并且m>1），每段绳子的长度记为 k[0],k[1]…k[m - 1] 。请问 k[0]k[1]…*k[m - 1] 可能的最大乘积是多少？例如，当绳子的长度是8时，我们把它剪成长度分别为2、3、3的三段，此时得到的最大乘积是18。答案需要取模 1e9+7（1000000007），如计算初始结果为：1000000008，请返回 1。

贪心，用long接收res

class Solution {
    public int cuttingRope(int n) {
        //需要取模的话就不能用动态规划了
        //贪心算法
        //尽可能地取3，最后余数可能是0，1，2
        //0：结果即为所有三相乘
        //1：取出一个三，2*2>3*1
        //2: 所有三相乘再*2
        if(n<1) return 0;
        if(n==1) return 1;
        if(n==2) return 1;
        if(n==3) return 2;

        int e=1000000007;
        //res一定要用long类型,不然大数会出错
        long res = 1;   
        if(n%3==2) {
            n-=2;
            res=2;
        }
        else if(n%3==1){
            n-=4;
            res = 4;
        }

        while(n>0){
            res*=3;
            res = res%e;
            n-=3;
        }
        return (int)res%e;
    }
}

14.数值的整数次方

实现 pow(x,n)，即计算 x 的 n 次幂函数（即，xn）。不得使用库函数，同时不需要考虑大数问题。

快速幂写法 x^n,用long接收n

class Solution {
    public double myPow(double x, int n) {
        //快速幂写法
        if(n==0&&x==0) return -1;
        else if(n==0) return 1;
        else if(x==0) return 0;
        //用long接收，防止越界
        long p = n;

        if(p<0){
            x = 1/x;
            p = -p;
        }    
        double res = 1.0;
        while(p>0){
            if((p&1)==1) res*=x;
            x *=x;
            p >>=1;

        }
        return res;
    }
}

16.删除链表的节点

class Solution {
    public ListNode deleteNode(ListNode head, int val) {
        if(head==null) return null;
        //先搜索再删除
        //dummy 哨兵节点
        ListNode dummy = new ListNode(0);
        dummy.next = head;
        //双指针移动
        ListNode pre = dummy;
        while(head!=null){

                pre.next = head.next;
            }
            pre = pre.next;
            head = head.next;
        }
        return dummy.next;
    }
}

17.正则式匹配

请实现一个函数用来匹配包含’. ‘和’‘的正则表达式。模式中的字符’.’表示任意一个字符，而’‘表示它前面的字符可以出现任意次（含0次）。在本题中，匹配是指字符串的所有字符匹配整个模式。例如，字符串”aaa”与模式”a.a”和”abaca”匹配，但与”aa.a”和”ab*a”均不匹配。

动态规划

讨论是不是*，取不取0

不是*: dp(i,j) = dp(i-1,j-1)，

取0: dp(i,j) = dp(i,j-2)

不取0: dp(i,j) = dp(i-1,j)

class Solution {
    public boolean isMatch(String s, String p) {
        //动态规划
        //dp[i][j]: 表示s的前i个和p的前j个相匹配，即0~i-1和0~j-1匹配
        int m = s.length();
        int n = p.length();
        boolean[][] dp = new boolean[m+1][n+1];

        for(int i=0;i<=m;i++){
            for(int j=0;j<=n;j++){
                //初始化 dp[0][0]=true,其余dp[i][0]=false;
                if(j==0){
                    dp[i][0] = (i==0);

                }else{
                    //如果不是*
                    if(p.charAt(j-1)!='*'){
                        //如果s[i-1]和p[i-1]相匹配，则前i个与前j个匹配情况同上一个相同
                        if(i>0&&(s.charAt(i-1)==p.charAt(j-1)||p.charAt(j-1)=='.')){
                            dp[i][j] = dp[i-1][j-1];
                        }
                    }
                    //如果是'*'
                    else{
                        //'*'若取零,则前ij个的匹配情况就和前i j-2个的匹配相同
                        if(j>=2){
                            dp[i][j] |= dp[i][j-2];
                        }
                        //'*'不取零,就要看前i j-1个是否相匹配
                        //s[i-1]和p[j-2]比较，相等则匹配情况与前i-1 j个相同，i继续往后匹配
                        if(j>=2&&i>0&&(s.charAt(i-1)==p.charAt(j-2)||p.charAt(j-2)=='.')){
                            dp[i][j] |= dp[i-1][j];
                        }
                    }
                }
            }
        }
        return dp[m][n];
    }
}

18.表示数值的字符串

请实现一个函数用来判断字符串是否表示数值（包括整数和小数）。

class Solution {
    public boolean isNumber(String s) {
        char[] ch = s.trim().toCharArray();
        if(ch==null||ch.length==0) return false;
        //设置标志位记录前面是否出现过该元素
        boolean e = false;
        boolean dot = false;
        boolean num = false;

        for(int i=0;i<ch.length;i++){
            if(ch[i]>='0'&&ch[i]<='9'){
                num=true;
            }
            else if(ch[i]=='e'||ch[i]=='E'){
                //如果是e, e前面一定有数字,不能有e,且e之后一定得有数字
                if(!num||e) return false;
                e = true;
                //重置num,便于判断后面有没有数字
                num = false;
            }
            else if(ch[i]=='.'){
                //.前面不能有e,不能有.
                if(dot||e) return false;
                dot=true;
            }
            else if(ch[i]=='+'||ch[i]=='-'){
                //符号出现在第一位或者e后面一位,否则不对
                if(i!=0&&ch[i-1]!='e'&&ch[i-1]!='E') return false;
            }
            else{
                return false;
            }
        }
        return num;
    }
}

19.调整数组顺序使奇数位于偶数前面

输入一个整数数组，实现一个函数来调整该数组中数字的顺序，使得所有奇数位于数组的前半部分，所有偶数位于数组的后半部分。

解1:暴力搜索,可以不改变相对顺序
解2:快排,但相对顺序可能发生改变

class Solution {
    public int[] exchange(int[] nums) {
        //快排
        if(nums==null||nums.length==0) return nums;
        int l = 0,r = nums.length-1;

        while(l<r){
            //r从右往左找奇数

            }

            //从下往上
            for(int i=down;i>=up;i--){

            }
            if(++left>right) break;
        }

    int[] temp;
    public int reversePairs(int[] nums) {
        //重写归并算法
        //一路二分，最后归并
        if(nums==null||nums.length<=0) return 0;
        temp = new int[nums.length];

        return mergeSort(nums,0,nums.length-1);
    }

    int mergeSort(int[] nums,int start,int end){
        if(start>=end) return 0;
        int l = start,r = end;
        //一路二分
        int mid = l+(r-l)/2;
        int leftc = mergeSort(nums,start,mid);
        int rightc = mergeSort(nums,mid+1,end);
        //归并
        int cross = merge(nums,start,mid,end);
        return cross+leftc+rightc;
    }

    int merge(int[] nums,int start,int mid,int end){

        //先存进临时数组
        for(int i=start;i<=end;i++){
            temp[i] = nums[i];
        }

        int l=start,r=mid+1;
        int count = 0;
        for(int index=start;index<=end;index++){
            //如果temp[l]的值比temp[r]大的话，l后面的都比它大，
            //这就存在mid-l+1个逆序对
            if(l>mid){
                nums[index] = temp[r++];
            }
            else if(r>end){
                nums[index] = temp[l++];
            }
            else if(temp[l]>temp[r]){
                nums[index] = temp[r++];
                count+=(mid-l+1);
            }else{
                nums[index] = temp[l++];
            }
        }
        return count;

    }
}

            }

        }

    }

    public int max_value() {
        //deq队首即为最大值
        return deq.size()==0?-1:deq.peekFirst();
    }

    public void push_back(int value) {
        //que正常存取
        que.add(value);

        //存入算法与滑动窗口最大值相同
        //只是窗口就是整个队列
        while(!deq.isEmpty()&&deq.peekLast()<value){
            deq.removeLast();
        }
        deq.add(value);
    }

    public int pop_front() {
        int v = que.size()>0?que.poll():-1;
        if(deq.size()>0&&v==deq.peekFirst()) deq.poll();
        return v;
    }
}

65.n个骰子的点数

把n个骰子扔在地上，所有骰子朝上一面的点数之和为s。输入n，打印出s的所有可能的值出现的概率。你需要用一个浮点数数组返回答案，其中第 i 个元素代表这 n 个骰子所能掷出的点数集合中第 i 小的那个的概率。

动态规划 dp [i] [j] 表示 i个骰子投出j点的次数

class Solution {
    public double[] dicesProbability(int n) {
        //n个骰子，有n~6n种点数 解的长度为6n-n+1
        //总数是6^n次
        //概率=dp[i][j]/6^n
        //dp[i][j]表示 i个骰子投出j点的次数
        double all = Math.pow(6,n);

        //动态规划
        double[] res = new double[5*n+1];
        int[][] dp = new int[n+1][6*n+1];

        //初始化
        dp[0][0] = 1;
        //遍历更新
        //n个骰子
        for(int i=1;i<=n;i++){
            //n~6n种点数
            for(int j=1;j<=6*n;j++){
                //最后一投1~6，j<6的时候k到j就够了
                for(int k = 1;k<=Math.min(j,6);k++){
                    dp[i][j] += dp[i-1][j-k];
                }
            }
        } 

        for(int i=n;i<=6*n;i++){
            res[i-n] = dp[n][i]/all;
        }
        return res;
    }
}

66.扑克牌中的顺子

从扑克牌中随机抽5张牌，判断是不是一个顺子，即这5张牌是不是连续的。2～10为数字本身，A为1，J为11，Q为12，K为13，而大、小王为 0 ，可以看成任意数字。A 不能视为 14。

class Solution {
    public boolean isStraight(int[] nums) {
        //1. 先排序
        Arrays.sort(nums);
        //2. 去掉0
        int l=0,r=nums.length;
        while(nums[l]==0) l++;
        //3. 有重复的指定不是
        for(int i=l;i<r-1;i++){
            if(nums[i]==nums[i+1]) return false;
        }
        //4. 再次前提下，只要max-min<=4 必是顺子
        return nums[r-1]-nums[l]<=4;

    }
}

67.圆圈中最后剩下的数字

0,1,···,n-1这n个数字排成一个圆圈，从数字0开始，每次从这个圆圈里删除第m个数字（删除后从下一个数字开始计数）。求出这个圆圈里剩下的最后一个数字。例如，0、1、2、3、4这5个数字组成一个圆圈，从数字0开始每次删除第3个数字，则删除的前4个数字依次是2、0、4、1，因此最后剩下的数字是3。

约瑟夫环

class Solution {
    public int lastRemaining(int n, int m) {
        //n    0,1,2,...,m-2, m,m+1,m+2,...n
        //n-1                 0,1,2,...n-m
        //n个数字，删掉一个后，n-1个数字的结果是同一个
        //但重新编号 存在对应的映射
        //f(n)=(f(n-1)+m)%n

        if(n==0) return 0;
        return (lastRemaining(n-1,m)+m)%n;

    }
}

68.股票的最大利润

假设把某股票的价格按照时间先后顺序存储在数组中，请问买卖该股票一次可能获得的最大利润是多少？

class Solution {
    public int maxProfit(int[] prices) {
        if(prices==null||prices.length==0) return 0;
        int res = 0;
        //i前的最小价格
        int minv = prices[0];
        for(int i=0;i<prices.length;i++){
            res = Math.max(res,prices[i]-minv);
            //更新i前的最小值
            minv = Math.min(minv,prices[i]);
        }
        return res;
    }
}

71.构建乘积数组

给定一个数组 A[0,1,…,n-1]，请构建一个数组 B[0,1,…,n-1]，其中 B[i] 的值是数组 A 中除了下标 i 以外的元素的积, 即 B[i]=A[0]×A[1]×…×A[i-1]×A[i+1]×…×A[n-1]。不能使用除法。

class Solution {
    public int[] constructArr(int[] a) {
        //两边分别乘好，再相乘
        int n=a.length;
        int[] b = new int[n];
        int p = 1;
        for(int i=0;i<n;i++){
            b[i] = p; //b[0] = 1  ,b[1] = a[0], b[2] = a[0]*a[1] 都是前半部分
            p *=a[i];
        }

        p = 1;
        //再循环乘一波后半部分
        for(int i = n-1;i>=0;i--){
            b[i] *=p; //b[n-1] = b[n-1]*1 b[n-2] = b[n-2]*a[n-1]从后往前乘上后半部分
            p*=a[i];
        }

        return b;
    }

}

72.把字符串转换成整数

写一个函数 StrToInt，实现把字符串转换成整数这个功能。不能使用 atoi 或者其他类似的库函数。首先，该函数会根据需要丢弃无用的开头空格字符，直到寻找到第一个非空格的字符为止。当我们寻找到的第一个非空字符为正或者负号时，则将该符号与之后面尽可能多的连续数字组合起来，作为该整数的正负号；假如第一个非空字符是数字，则直接将其与之后连续的数字字符组合起来，形成整数。该字符串除了有效的整数部分之后也可能会存在多余的字符，这些字符可以被忽略，它们对于函数不应该造成影响。注意：假如该字符串中的第一个非空格字符不是一个有效整数字符、字符串为空或字符串仅包含空白字符时，则你的函数不需要进行转换。在任何情况下，若函数不能进行有效的转换时，请返回 0。

class Solution {
    public int strToInt(String str) {
        char[] ch = str.trim().toCharArray();
        if(ch==null||ch.length==0) return 0;
        int sign = 1;
        int index = 1;       
        //用于判断最大值
        int temp = 214748364;
        if(ch[0]=='-') sign=-1;
        else if(ch[0]!='+') index = 0;

        int res = 0;       
        for(int i=index;i<ch.length;i++){
            //不是数字，直接返回
            if(ch[i]<'0'||ch[i]>'9') break;
            //超过边界，返回最大值
            if(res>temp ||(res==temp&&ch[i]>'7')){
                return sign==1?Integer.MAX_VALUE:Integer.MIN_VALUE;
            }
            //拼接上数字
            res = res*10+ch[i]-'0';
        } 
        return res*sign;
    }
}

73.二叉搜索树的最近公共祖先

        //递归
        //如果p>r且q>r 说明都在r右边 向右搜索
        //如果p<r且q<r 说明都在r左边 向左搜索
        if(p.val>root.val&&q.val>root.val) return lowestCommonAncestor(root.right,p,q);
        if(p.val<root.val&&q.val<root.val) return lowestCommonAncestor(root.left,p,q);
        return root;
    }
}

74.二叉树的最近公共祖先

给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        //递归

        //向下搜索 如果都在左边说明公共节点在左边 如果都在右边说明公共节点在右边

        //递归终止条件：返回null或者找到了p或q
        if(root==null||root==p||root==q) return root;

        //搜索左边有没有p或q
        TreeNode left = lowestCommonAncestor(root.left,p,q);
        //搜索右边有没有p或q
        TreeNode right = lowestCommonAncestor(root.right,p,q);

        //如果left==null 说明都在右边 继续搜索右边
        if(left==null) return right;
        //如果right==null 说明都在左边 继续搜索左边
        if(right==null) return left;

        //左右都不为空 说明pq在root的一左一右 此时root即为公共节点
        return root;

    }
}

public class Solution {
    public ListNode deleteDuplication(ListNode pHead) {

        //双指针
        //新建一个哨兵节点
        ListNode dummy = new ListNode(0);
        dummy.next = pHead;
        ListNode pre = dummy;
        ListNode cur = pHead;

        while(cur!=null){

            if(cur.next != null && cur.val == cur.next.val){
                //如果cur.val==cur.next.val cur一路后移
                //cur一直移到不相等
                while(cur.next != null && cur.val == cur.next.val){
                    cur = cur.next;
                }
                //此时，cur已经到了最后一个此值的重复节点，后移判断下一个值
                cur = cur.next;
                pre.next = cur;
            }
            else{
                //没有重复节点了
                //双指针后移
                pre = cur;
                cur = cur.next;

            }
        }

        return dummy.next;
    }
}
